## Subarrays with equal 1s and 0s

Given an array containing 0s and 1s. Find the number of subarrays having equal number of 0s and 1s.

#### Example 1:

```
Input:
N = 7
A[] = {1,0,0,1,0,1,1}
Output: 8
Explanation: The index range for the 8
sub-arrays are: (0, 1), (2, 3), (0, 3), (3, 4),
(4, 5) ,(2, 5), (0, 5), (1, 6)
```

#### Example 2:

```
Input:
N = 5
A[] = {1,1,1,1,0}
Output: 1
Explanation: The index range for the
subarray is (3,4).
```

#### Your Task:

You don't need to read input or print anything. Your task is to complete the function `countSubarrWithEqualZeroAndOne()` which takes the array arr[] and the size of the array as inputs and returns the number of subarrays with equal number of 0s and 1s.

Expected Time Complexity: `O(N)`.  
Expected Auxiliary Space: `O(N)`.

```
Constraints:
1 <= N <= 10^6
0 <= A[i] <= 1
```

```c++
long long int countSubarrWithEqualZeroAndOne(int arr[], int n)
{
    for(int i=0; i<n; i++) {
        if(arr[i] == 0)
            arr[i] = -1;
    }
    int sum = 0;
    unordered_map<int, int> prefixSum;
    int count = 0;
    int temp = arr[0];
    prefixSum[arr[0]]++;
    if(arr[0] == sum) count++;
    for(int i=1; i<n; i++) {
        temp = temp + arr[i];
        if(temp == sum)
            count++;
        if(prefixSum.find(temp-sum) != prefixSum.end())
            count += prefixSum[temp-sum];
        prefixSum[temp]++;
    }
    return count;
}
```
